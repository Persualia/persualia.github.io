Guía de Personalización de Temas

06 de August, 2025

Por Equipo de Diseño

Una de las características más poderosas de Persualia es su sistema de temas altamente personalizable. En esta guía, te mostraremos cómo sacar el máximo provecho de esta funcionalidad.

Sistema de Temas

Persualia utiliza DaisyUI 5 para su sistema de temas, lo que nos proporciona:

  • Más de 30 temas predefinidos
  • Fácil personalización
  • Cambio dinámico de temas
  • Persistencia en localStorage

Temas Disponibles

Temas Principales

  • Persualia: Nuestro tema personalizado con colores corporativos
  • Light: Tema claro y minimalista
  • Dark: Tema oscuro para mejor experiencia nocturna
  • Corporate: Diseño profesional para uso empresarial

Temas Creativos

  • Synthwave: Estética retro-futurista
  • Cyberpunk: Diseño futurista y vibrante
  • Cupcake: Colores suaves y pasteles
  • Halloween: Tema temático naranja y negro

Cómo Cambiar Temas

Método 1: Selector de Temas

  1. Busca el icono de sol/luna en la barra de navegación
  2. Haz clic para abrir el menú de temas
  3. Selecciona tu tema favorito
  4. ¡El cambio es instantáneo!

Método 2: Programáticamente

// Cambiar tema mediante JavaScript
function setTheme(themeName) {
  document.documentElement.setAttribute('data-theme', themeName);
  localStorage.setItem('theme', themeName);
}

// Ejemplos
setTheme('dark');
setTheme('cyberpunk');
setTheme('persualia');

Personalización Avanzada

Creando tu Propio Tema

Puedes crear temas personalizados modificando el archivo tailwind.config.js:

daisyui: {
  themes: [
    {
      miTemaPersonalizado: {
        "primary": "#1e40af",
        "secondary": "#7c3aed", 
        "accent": "#f59e0b",
        "neutral": "#3d4451",
        "base-100": "#ffffff",
        "info": "#3abff8",
        "success": "#36d399",
        "warning": "#fbbd23",
        "error": "#f87272",
      },
    },
    // otros temas...
  ],
}

Variables CSS Personalizadas

También puedes usar variables CSS para personalización en tiempo real:

:root {
  --primary-color: #0ea5e9;
  --secondary-color: #7c3aed;
  --accent-color: #f59e0b;
}

.mi-componente {
  background: var(--primary-color);
  border: 1px solid var(--secondary-color);
}

Mejores Prácticas

Accesibilidad

  • Siempre verifica el contraste de colores
  • Prueba tus temas con usuarios con discapacidades visuales
  • Considera la legibilidad en diferentes condiciones de luz

Rendimiento

  • Evita animaciones excesivas en los cambios de tema
  • Optimiza las transiciones CSS
  • Mantén los temas ligeros en tamaño

Experiencia de Usuario

  • Proporciona una previsualización antes de aplicar el tema
  • Mantén la consistencia en la iconografía
  • Ofrece temas para diferentes momentos del día
