Nuestra Misión

Persualia es una aplicación web moderna que combina las mejores tecnologías actuales para ofrecer una experiencia de usuario excepcional. Construida con Jekyll, React, TailwindCSS v4.1 y DaisyUI 5 con el elegante tema persualia, representa el equilibrio perfecto entre rendimiento, diseño y funcionalidad.

Tecnologías Utilizadas

Jekyll

Jekyll es un generador de sitios estáticos que nos permite crear sitios web rápidos y seguros. Con su sistema de plantillas Liquid y soporte para Markdown, facilita la creación y mantenimiento de contenido.

React

React nos proporciona la capacidad de crear interfaces de usuario interactivas y dinámicas. Utilizamos React para componentes específicos que requieren interactividad en el lado del cliente.

TailwindCSS v4.1

TailwindCSS es nuestro framework de CSS utility-first que nos permite crear diseños personalizados rápidamente sin salir del HTML. La versión 4.1 incluye las últimas mejoras en rendimiento y nuevas utilidades.

DaisyUI 5 con Tema persualia

DaisyUI complementa TailwindCSS con componentes semánticos predefinidos. Utilizamos el tema persualia con personalizaciones avanzadas que proporcionan una experiencia visual elegante y minimalista.

El Tema persualia

Nuestro sitio utiliza el tema persualia de DaisyUI 5 con personalizaciones únicas que incluyen:

El tema persualia proporciona una base elegante y minimalista perfecta para experiencias web modernas.
  • Presets personalizados: Arctic, Ice Purple, Pine y Storm
  • Efectos especiales: Frost, Glass y gradientes animados
  • Componentes únicos: Botones con efectos persualia y cards con glassmorphism
  • Tokens personalizables: Fácil personalización de colores y formas
